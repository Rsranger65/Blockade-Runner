Building Directions
-------------------

1. Add `core.jar` from **Processing 2.2.1** to the classpath
2. Compile as a runnable `.jar` file with **JDK 1.8**
3. Place the `res` folder next to the `.jar` file (Launch path: net.kopeph.ld31.LD31)
